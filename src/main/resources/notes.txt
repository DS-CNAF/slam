Create team

curl -i -X POST -H "Content-Type:application/json" -H "Cookie:JSESSIONID=$JSESSIONID" -d '{ "name" : "ninjas" }' http://localhost:8080/teams

Create role

curl -i -X POST -H "Content-Type:application/json" -H "Cookie:JSESSIONID=$JSESSIONID" -d '{ "name" : "watcher" }' http://localhost:8080/roles

Add user to team

curl -i -X POST -H "Content-Type:text/uri-list" -H "Cookie:JSESSIONID=$JSESSIONID" -d 'http://localhost:8080/users/2' http://localhost:8080/teams/3/members

Add role

curl -i -X POST -H "Content-Type:text/uri-list" -H "Cookie:JSESSIONID=$JSESSIONID" -d 'http://localhost:8080/roles/3' http://localhost:8080/users/153/roles
 
Change affiliation state  
 
curl -i -X PATCH -H "Content-Type:application/json" -H "Cookie:JSESSIONID=$JSESSIONID" -d '{ "status" : "ACTIVE" }' http://localhost:8080/affiliations/8
